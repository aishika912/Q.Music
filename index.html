Parts.io

Electronic Component Icons

Introduction

A custom SVG font created for parts.io icons using Font Custom. Get all the advantages that come with FontCustom fonts such as small file sizes, flexible icon sizes, and automatic regeneration with command line watching. Icons available for most categories of hardware components including (but not limited to) capacitors, resistors, sensors, and programmable logic.

Installing

  • Install Homebrew

    ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  • Download Hardware Icon SVGs and place in project directory

  • Install Font Custom

    brew install fontforge eot-utils ttfautohilt
gem install fontcustom

  • Run Font Custom

    fontcustom watch /path/to/vectors

  • Include Font Custom CSS in project

    <link href="path/to/fontcustom/fontcustom.css" rel="stylesheet">

Usage

  • Add icon to selector

    <span class="icon-active-filters" style="font-size:120px;"></span>

  • Result

  • Style as needed

    <span class="icon-fiber-optics my-class"></span>
    .my-class {
   color: blue;
   font-size: 90px;
}

  • Result
